Addon Package for Elementor. Includes FlexSlider (Carousel, Basic) + WooCommerce Slider (Products & Categories)

== Description ==
This lightweight Plugin (Addon for Elementor) allows you to use FlexSlider in Elementor (Full Responsive).

<h3>Features (FREE)</h3>
<h4>Basic FlexSlider</h4>
<ul>
<li>Image Slider</li>
<li>Image Slider + Thumbnail Navigation</li>
<li>Carousel</li>
</ul>
>Options:
Animation: Slide/Fade<br />
Slideshow: Yes/No<br />
Loop: Yes/No<br />
Touch Support: Yes/No<br/>
Animation Speed: 600-5000ms<br/>
ControlNav: Yes/No<br/>
+ a lot options about styling
<br />
<h4>FlexSlider WooCommerce</h4>
<ul>
<li>WooCommerce Product Carousel<br />
Type: All Products or Featured Products<br />
</li>
<li>WooCommerce Categorie Carousel<br />
Type: All / Selected or child Categories of Parent Categories<br />
</li>
</ul>
>Options:
Elements in Line: 2-6<br />
Slideshow: Yes/No<br />
Loop: Yes/No<br />
Touch: Yes/No<br />
Animation Speed: 600-5000ms<br/>
+ a lot options about styling
<br />
<h4>FlexSlider Post Slider / Carousel</h4>
<ul>
<li>Post Slider with Thumbnail Navigation<br />
Type: Specified Posts or Posts from selected Categories<br />
</li>
<li>Post Carousel<br />
Type: Specified Posts or Posts from selected Categories<br />
</li>
</ul>

== Installation ==
1. Upload plugin over wordpress (Or install the plugin via the WordPress build in function)
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Edit your desired page via "Elementor" and type in to the search field "FlexSlider".
